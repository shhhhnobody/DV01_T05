Notes:

Scope & Time Series:

  • The value of fines per 10k license holder is obtained as such:
  • Fines per 10,000 licences = (Number of speeding fines ÷ Number of licensed drivers) × 10,000
  • Using “per 10,000 licences” makes the comparison fair because each state has a different number of licensed drivers.
  • For example, NSW has millions of licence holders, while NT has fewer than 200,000 — so using raw numbers alone would be misleading.
  • This metric helps compare jurisdictions on an equal basis.This is very important for fairness, and used officially by the Australian Road Safety Data Hub (BITRE).
